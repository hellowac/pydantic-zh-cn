# pydantic-zh-cn

使用 Python 类型提示进行数据验证 - 中文版文档

## 包管理工具

参考: <https://pdm.fming.dev/latest/>

这里使用pdm

初始化:

```powershell
pdm init
```

添加包:

```powershell
pdm add <package name>
```
